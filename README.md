# Прогулкин 🚶

**Прогулкин** — приложение для трекинга прогулок, которое превращает обычную ходьбу в увлекательный процесс исследования мира и поиска единомышленников по интересам.

## Концепция

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   "Превращаем прогулки в приключения,                       │
│    находим единомышленников по интересам"                   │
│                                                             │
│   ┌─────────┐     ┌─────────┐     ┌─────────┐              │
│   │ Гуляем  │ ──→ │Отмечаем │ ──→ │Находим  │              │
│   │         │     │интересное│    │ друзей  │              │
│   └─────────┘     └─────────┘     └─────────┘              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

Пользователи могут создавать и находить различные объекты на карте во время прогулок:
- **👹 Мусорные монстры** — отмечаем мусор, убираем, получаем очки
- **📍 Заметки об интересных местах** — делимся находками, находим единомышленников
- **📜 Секретные сообщения** — послания, которые читаются только в определённом месте
- **🔔 Напоминалки** — персонажи, которые напоминают о делах в нужном месте
- **🦊 Существа** — коллекционные цифровые существа из русской мифологии

## Ключевая особенность: Связи через интересы

Не просто "гулять с теми, кто рядом", а находить людей с похожими интересами:

```
Примеры:
🐿️ "Здесь утром белки берут орехи из рук"
   → Фотографы, любители природы

📚 "Библиотека с редкими книгами по математике"
   → Студенты, книголюбители

📸 "Лучшая точка для съёмки заката"
   → Фотографы
```

Когда кому-то интересна ваша заметка — можно показать контакт и начать общение.

## Децентрализация данных

### Философия приватности

**Прогулкин** не хранит данные пользователей на серверах. Все данные находятся на устройствах пользователей и синхронизируются напрямую между ними.

```
┌─────────────────────────────────────────────────────────────┐
│                      Прогулкин (P2P)                        │
│                                                             │
│   Пользователь A ←──P2P──→ Пользователь B                  │
│         ↑                          ↑                        │
│         │                          │                        │
│         └──────── Signaling ───────┘                       │
│                  (только знакомит)                          │
│                                                             │
│   Данные хранятся только на устройствах пользователей      │
└─────────────────────────────────────────────────────────────┘
```

### Что хранится и где

| Где | Что хранится |
|-----|--------------|
| **Устройство пользователя** | Маршруты, объекты, фото, сообщения, настройки |
| **Signaling сервер** | Временный ID устройства, зона (geohash), IP:порт |
| **Нигде** | Имена, email, пароли, личные данные |

### Преимущества подхода

- ✅ **Полная приватность** — ваши данные только у вас
- ✅ **Нет регистрации** — не нужен email или пароль
- ✅ **Работает офлайн** — все функции доступны без интернета
- ✅ **Нет цензуры** — децентрализованная сеть не контролируется
- ✅ **Долгосрочность** — данные не исчезнут при закрытии сервиса

## Объекты на карте

### 👹 Мусорные монстры

Превращаем проблему мусора в игру. Каждый мусорный объект — это «монстр», которого нужно «победить» (убрать).

| Класс | Сложность | Примеры | Очки |
|-------|-----------|---------|------|
| 🟢 1 | Взять и нести | Бумажка, бутылка | 10 |
| 🟡 2 | Нужен пакет | 3-5 бутылок | 20 |
| 🟠 3 | Нужны инструменты | Шины, техника | 30 |
| 🔴 4 | Нужна помощь | Мебель | 40 |
| 💀 5 | Нужна организация | Свалка | 50 |

**Типы мусора:** 🍺 Бутылки, 🗞️ Бумага, 🛞 Шины, 🛋️ Мебель, 🏗️ Строймусор, 📺 Электроника, 🥤 Пластик, 🍂 Органика

### 📍 Заметки об интересных местах 🆕

Отмечайте интересные места и находите единомышленников:

| Категория | Примеры |
|-----------|---------|
| 🐿️ Природа | "Здесь белки берут орехи из рук" |
| 📚 Культура | "Библиотека с редкими книгами" |
| 📸 Фото | "Лучшая точка для заката" |
| 🏃 Спорт | "Удобные турники в тени" |
| ☕ Еда | "Отличная кофейня для работы" |

**Механика:**
1. Создаёте заметку с фото и описанием
2. Другие пользователи видят её на карте
3. Если интересно — ставят "Интересно" 👍
4. Вы получаете уведомление
5. Можно показать контакт для связи

### 🔔 Напоминалки-персонажи 🆕

Игровые персонажи, которые напоминают о делах в нужном месте:

| Персонаж | Пример напоминания |
|----------|-------------------|
| 🐻 Копатыч | "Купить флюс ЛТИ-120 в аптеке" |
| 🐰 Крош | "Здесь живёт друг, можно зайти" |
| 🦔 Ёжик | "Опустить письмо" |

**Механика:** Ставите метку с персонажем, указываете радиус (10-500м). Когда подходите — получаете уведомление!

### 📜 Секретные сообщения

Сообщения, которые можно прочитать только находясь в определённом месте.

**Типы:** 📜 Текст, ❓ Загадка, 📖 История, 💫 Пожелание, 💡 Совет, 💭 Воспоминание

**Механика:**
1. Создаёте секрет в определённом месте
2. Указываете радиус разблокировки (10-200м)
3. Другие видят только заголовок
4. Чтобы прочитать — нужно прийти на место

### 🦊 Существа русской мифологии

Коллекционные цифровые существа, появляющиеся в определённых местах.

| Редкость | Существа |
|----------|----------|
| ⚪ Обычные | 🏠 Домовой, 🐠 Золотая рыбка |
| 🟢 Необычные | 🌿 Кикимора, 🧜‍♀️ Русалка |
| 🔵 Редкие | 🌲 Леший, 🌊 Водяной |
| 🟣 Эпические | 🧙‍♀️ Баба Яга, 🦅 Сирин |
| 🟡 Легендарные | 🐉 Змей Горыныч, 🔥 Жар-птица |

## Возможности

- 📍 **Запись маршрута** — отслеживание GPS позиции с фильтрацией выбросов

## Установка

### Требования

- Flutter SDK 3.0+
- Android SDK 21+ (Android 5.0+)

### Быстрый старт

```bash
# Клонирование репозитория
git clone https://github.com/NikasAl/progulkin.git
cd progulkin

# Установка зависимостей
flutter pub get

# Запуск на устройстве
flutter run
```

### Сборка

```bash
# Сборка APK (Android)
flutter build apk --release

# Сборка App Bundle (для RuStore / Google Play)
flutter build appbundle --release
```

## Настройка P2P

По умолчанию приложение использует публичный signaling сервер. Для запуска своего:

```bash
# На сервере с публичным IP
dart run bin/signaling_server.dart --port 9000

# В приложении укажите адрес вашего сервера
# (в настройках P2P или конфигурации)
```

## Разрешения

### Android

Добавлены в `AndroidManifest.xml`:
- `ACCESS_FINE_LOCATION` — точная геолокация
- `ACCESS_COARSE_LOCATION` — примерная геолокация
- `ACCESS_BACKGROUND_LOCATION` — геолокация в фоне
- `ACTIVITY_RECOGNITION` — распознавание активности
- `INTERNET` — P2P синхронизация и карта

## Безопасность контента

- ✅ **Фото с модерацией** — подтверждаются пользователями
- ✅ **Локальный фильтр** — запрещённые слова блокируются
- ✅ **Репутация** — объекты новичков скрыты по умолчанию
- ✅ **Голосование** — пользователи подтверждают или опровергают объекты
- ✅ **Автоскрытие** — объекты с жалобами скрываются

## Политика конфиденциальности

Для магазинов приложений (RuStore, Google Play):

```
1. Все данные хранятся на устройстве пользователя
2. P2P синхронизация — пользователь контролирует
3. Нет регистрации и личных данных
4. Signaling сервер только знакомит устройства
5. Можно отключить синхронизацию в настройках
```
