# Прогулкин 🚶

Flutter приложение для трекинга прогулок с отображением маршрутов на карте OpenStreetMap и подсчётом шагов.

## Возможности

- 📍 **Запись маршрута** - отслеживание GPS позиции во время прогулки
- 🗺️ **OpenStreetMap** - бесплатная карта без необходимости API ключа
- 👟 **Подсчёт шагов** - использование нативного шагомера или акселерометра
- 📊 **Статистика** - расстояние, время, средняя скорость
- 💾 **История прогулок** - сохранение всех прогулок локально
- 🌙 **Тёмная тема** - автоматическое переключение темы
- 🔒 **Приватность** - никаких API ключей и внешних зависимостей для карт

## Скриншоты

Главный экран показывает карту с текущим маршрутом, панель статистики (расстояние, время, скорость) и счётчик шагов.

## Установка

### Требования

- Flutter SDK 3.0+
- Android SDK 21+ / iOS 12+

### Быстрый старт

```bash
# Клонирование репозитория
git clone https://github.com/NikasAl/progulkin.git
cd progulkin

# Установка зависимостей
flutter pub get

# Запуск на устройстве
flutter run
```

### Сборка

```bash
# Сборка APK (Android)
flutter build apk --release

# Сборка App Bundle (Android, для Google Play)
flutter build appbundle --release

# Сборка для iOS (требуется macOS)
flutter build ios --release
```

## Конфигурация

Параметры приложения находятся в `lib/config/app_config.dart`:

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| `trackingDistanceFilter` | Мин. расстояние между точками (м) | 5 |
| `averageStepLength` | Средняя длина шага (м) | 0.75 |
| `minStepIntervalMs` | Мин. интервал между шагами (мс) | 250 |
| `stepDetectionThreshold` | Порог детекции шага | 25.0 |
| `enableLogging` | Включить логирование | true |

## Разрешения

### Android

Добавлены в `AndroidManifest.xml`:
- `ACCESS_FINE_LOCATION` - точная геолокация
- `ACCESS_COARSE_LOCATION` - примерная геолокация
- `ACCESS_BACKGROUND_LOCATION` - геолокация в фоне
- `ACTIVITY_RECOGNITION` - распознавание активности (шагомер)

### iOS

Требуется добавить в `Info.plist`:
```xml
<key>NSLocationWhenInUseUsageDescription</key>
<string>Для записи маршрута прогулки</string>
<key>NSLocationAlwaysAndWhenInUseUsageDescription</key>
<string>Для записи маршрута прогулки в фоне</string>
<key>NSMotionUsageDescription</key>
<string>Для подсчёта шагов</string>
```

## Функции

### Запись маршрута
- Автоматическая фильтрация GPS-выбросов
- Игнорирование точек с низкой точностью
- Расчёт расстояния по формуле Гаверсинуса

### Подсчёт шагов
- Использование нативного шагомера (если доступен)
- Fallback на акселерометр при отсутствии шагомера
- Расчёт пройденного расстояния по количеству шагов

### История прогулок
- Сохранение в SharedPreferences
- Просмотр деталей каждой прогулки
- Карта маршрута на экране деталей
- График скорости по маршруту

## Разработка

### Код-стайл

Проект следует рекомендациям [Effective Dart](https://dart.dev/guides/language/effective-dart).

### Архитектура

Приложение построено на паттерне Provider для управления состоянием:
- `WalkProvider` - управление прогулками (создание, остановка, история)
- `PedometerProvider` - управление шагомером

Сервисы инкапсулируют работу с внешними API:
- `LocationService` - GPS трекинг с фильтрацией выбросов
- `PedometerService` - подсчёт шагов
- `StorageService` - сохранение данных

## Известные ограничения

- На некоторых устройствах Android может не работать нативный шагомер (используется акселерометр)
- GPS точность зависит от устройства и окружения
- Карты требуют интернет-соединения для загрузки тайлов
