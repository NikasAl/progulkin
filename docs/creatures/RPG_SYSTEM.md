## Философия дизайна

### Основной принцип

Приложение "Прогулкин" стремится нести **не агрессивный** посыл. Большинство существ из славянского фольклора не являются изначально враждебными — с ними можно договориться, подружиться или приручить.

### Классификация по характеру

| Категория | Описание | Примеры |
|-----------|----------|---------|
| **Мирные** | Приручаются без боя с помощью даров | Домовой, Леший, Русалка, Золотая рыбка |
| **Условно-враждебные** | Можно задобрить или вступить в бой | Кикимора, Кощей |
| **Враждебные** | Бой неизбежен | Змей Горыныч |

### Ключевые принципы

1. **Мирное сосуществование** — большинство существ можно приручить без боя
2. **Взаимовыгодный обмен** — существа ценят дары и отвечают дружбой
3. **Временные отношения** — приручённые существа остаются на карте, но могут уйти
4. **Естественный цикл** — неприручённые существа исчезают после прогулки

## Механика приручения

### Формула успеха

```
Шанс успеха = 
    Базовый шанс (от редкости) ×
    (1 + Бонус подарка) ×
    (1 + Бонус уровня игрока) ×
    (1 + Бонус артефактов)
```

### Пример расчёта

```
Приручение Лешего (редкий, базовый шанс 45%):
- Подарок: Кедровые орехи (+40%)
- Уровень игрока: 5 (+5%)
- Артефакт: Перо Жар-птицы (+30%)

Шанс = 45% × (1 + 0.40) × (1 + 0.05) × (1 + 0.30)
      = 45% × 1.40 × 1.05 × 1.30
      = 85.5%
```

### Процесс приручения

```
1. Встреча с существом
       ↓
2. Выбор "Предложить дар"
       ↓
3. Выбор предмета из инвентаря
       ↓
4. Расчёт шанса успеха
       ↓
5. ┌─────────────────────────┐
   │ Успех → Существо приручено │
   │ Провал → Существо убежало  │
   └─────────────────────────┘
```

### Альтернативные пути

**Для условно-враждебных:**

| Существо | Мирный путь | Боевой путь |
|----------|-------------|-------------|
| Кикимора | Болотные травы (50-70%) | Бой с ослаблением (60-75%) |
| Кощей | Найти яйцо (100%) | Бой (5-10%) |

**Для враждебных:**
- Только бой
- Награды выше

## Система дружбы

### Уровни дружбы

| Уровень | Название | Бонус | Время до ухода |
|---------|----------|-------|----------------|
| 1 | Знакомый | Нет | 3 дня |
| 2 | Друг | +5% очков | 7 дней |
| 3 | Лучший друг | +10% очков | 14 дней |
| 4 | Неразлучный | +15% очков + способность | 30 дней |
| 5 | Родная душа | +20% очков + 2 способности | Никогда не уходит |

### Повышение уровня дружбы

```
Способы повышения:
- Визит к существу: +5%
- Подарок: +10-30%
- Совместная прогулка: +15%
```

### Уход существа

```
Приручённое существо уходит если:
- Не навещали N дней (зависит от уровня дружбы)
- Уровень дружбы упал до 0

Предупреждение:
- За 1 день до ухода — уведомление
- При визите — счётчик сбрасывается
```

```
Принцип баланса:
- Средняя прогулка (30 мин) ≈ 100-150 очков
- Обычный дар ≈ 10-20 очков
- Хороший дар ≈ 50-100 очков (1-2 прогулки)
- Редкий дар ≈ 200+ очков (накопление)
```
