## Полное описание

**Прогулкин** — приложение для трекинга прогулок, которое превращает обычную ходьбу в увлекательный процесс исследования окружающего мира.

### Основные возможности:

🏃 **Трекинг прогулок**
- Запись маршрута на карте OpenStreetMap
- Подсчёт шагов через нативный шагомер
- Статистика: расстояние, время, скорость
- История всех прогулок

🗺️ **Объекты на карте**
- 🗑️ Мусорные монстры — отмечайте мусор и убирайте его
- 📜 Секретные сообщения — геокешинг для друзей
- 🦊 Существа русской мифологии — ловите Домовых, Лешего, Русалок и других!
- 📝 Заметки о местах — делитесь интересными находками
- 🔔 Напоминалки — гео-напоминания с Смешариками

🔒 **Приватность**
- Все данные хранятся на устройстве
- Нет регистрации и авторизации
- P2P-синхронизация без сервера
- Работает полностью офлайн

### Особенности:
- Не требует интернет для основных функций
- Кэширование карт для офлайн-режима
- Тёмная и светлая тема
- Расширенная статистика прогулок
- Экспорт/импорт данных

Отправляйтесь на прогулку, исследуйте мир и улучшайте его вместе с Прогулкиным!

## Сборка релиза

### Требования
- Flutter 3.38.9
- JDK 17
- Android SDK 36

### Создание keystore

```bash
mkdir -p android/keystore
keytool -genkey -v -keystore android/keystore/progulkin.jks \
  -keyalg RSA -keysize 2048 -validity 10000 \
  -alias progulkin
```

### Настройка подписи

1. Скопируйте `android/key.properties.example` в `android/key.properties`
2. Заполните пароли от keystore
3. Не коммитьте `key.properties`!

### Сборка

```bash
# Очистка
flutter clean

# Получение зависимостей
flutter pub get

# Сборка release APK
flutter build apk --release

# Сборка App Bundle (рекомендуется для RuStore)
flutter build appbundle --release
```

### Результат

- APK: `build/app/outputs/flutter-apk/app-release.apk`
- AAB: `build/app/outputs/bundle/release/app-release.aab`
