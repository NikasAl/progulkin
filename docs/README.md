## Краткий обзор RPG-системы

### Типы существ по характеру

| Категория | Примеры | Взаимодействие |
|-----------|---------|----------------|
| Мирные | Домовой, Леший, Русалка | Приручение дарами |
| Условно-враждебные | Кикимора, Кощей | Дары или бой |
| Враждебные | Змей Горыныч | Только бой |

### Редкость

```
⚪ Обычный → 🟢 Необычный → 🔵 Редкий → 🟣 Эпический → 🟡 Легендарный → 🔴 Мифический
```

### Экономика

- **Очки** — зарабатываются за прогулки
- **Дары** — покупаются за очки или через донат
- **Артефакты** — постоянные бонусы
