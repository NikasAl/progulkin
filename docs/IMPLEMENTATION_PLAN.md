# План реализации новых функций

## Обзор

Этот документ описывает план реализации новых функций для приложения Прогулкин с учётом обновлённой концепции "социальные связи через интересы".

---

## Приоритеты

| Приоритет | Обозначение | Описание |
|-----------|-------------|----------|
| 🔴 P0 | Критический | Блокирует другие задачи или важен для пользователей |
| 🟡 P1 | Высокий | Важная функция, нужна в ближайшее время |
| 🟢 P2 | Средний | Желательная функция |
| ⚪ P3 | Низкий | Можно отложить |

## Этап 1: Исправления и улучшения (P0)

### 1.1. Статистика объектов в прогулке

**Проблема:** Пользователь хочет видеть в статистике прогулки информацию о взаимодействии с объектами.

**Решение:** Добавить в WalkStatsPanel блок с объектной статистикой.

**Задачи:**
- [ ] Добавить поля в Walk модель для отслеживания объектов
- [ ] Обновить WalkStatsPanel для отображения:
  - Добавленных объектов
  - Подтверждённых объектов  
  - Опровергнутых объектов
  - Собранных очков

**Файлы:**
- `lib/models/walk.dart`
- `lib/screens/home/walk_stats_panel.dart`
- `lib/providers/walk_provider.dart`

**Оценка:** 2-3 часа

```dart
enum InterestCategory {
  nature,    // 🐿️ Природа
  culture,   // 📚 Культура
  sport,     // 🏃 Спорт
  food,      // ☕ Еда
  photo,     // 📸 Точки съёмки
  art,       // 🎨 Творчество
  games,     // 🎮 Игры
  tip,       // 💡 Совет
  other,     // ❓ Другое
}

class InterestNote extends MapObject {
  final InterestCategory category;
  final String title;
  final String description;
  final List<String> photoIds;         // ID фото в хранилище
  final int interestCount;
  final List<String> interestedUserIds;
  final bool contactVisible;
}
```
